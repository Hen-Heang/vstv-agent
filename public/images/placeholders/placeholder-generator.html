Replace these placeholder images with your actual property photos and agent headshots:

1. Property Images

Add your property photos to: public/images/properties/featured/

  • Use high-quality, well-lit photos
  • Recommended size: 1200x800px or larger
  • Format: JPG or WebP
  • Name files descriptively (e.g., luxury-condo-bkk1.jpg)

2. Agent Photos

Add agent headshots to: public/images/agents/

  • Professional headshots
  • Square aspect ratio (400x400px)
  • Consistent lighting and style
  • Name files with agent names (e.g., john-doe.jpg)

3. Company Images

Add company assets to: public/images/company/

  • Logo in SVG format (preferred)
  • Office building photos
  • Team photos
